# Let’s UI Icons

Let’s UI Icons is an open-source icon library designed to work seamlessly with the Let’s UI design system.
It provides clean, consistent, and scalable SVG icons, optimized for modern web projects.

## Usage

```html
<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" type="text/css" href="lets-ui-icons.css" />
  </head>
  <body>
    <i class="lui lui-caret-right"></i>
    <i class="lui lui-caret-left"></i>
  </body>
</html>
```

> [!NOTE]
> The library will soon be available via CDN, making it easy to include in any project.

## Weights

All icons are available in two styles: `solid` and `outline`. The style is defined by the base class: `lui` is the default and loads outline icons, while `lui-solid` is used exclusively for solid icons.

```html
<!-- Outline icons -->
<i class="lui lui-circle-check"></i>

<!-- Solid icons -->
<i class="lui-solid lui-circle-check"></i>
```
